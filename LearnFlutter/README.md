# LearnFlutter

Flutter / Dart 学习仓库，与 LearnPython 同系列，适合有移动端开发经验的开发者。

- **学习路线**：见 [学习路线.md](学习路线.md)
- **代码示例**：按章节放在 `01_Dart基础`、`02_Flutter基础` 等目录下

## 快速开始

```bash
# 确保已安装 Flutter
flutter doctor

# 运行 Dart 示例（不依赖 Flutter 项目，需在 LearnFlutter 目录下执行）
cd LearnFlutter
dart run 01_Dart基础/01_变量和数据类型.dart
```
